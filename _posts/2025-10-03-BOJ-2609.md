## 링크
[2609번: 최대공약수와 최소공배수](https://www.acmicpc.net/problem/2609)

## 풀이
<span class="txt_bg">유클리드 호제법</span>으로 최대공약수를 구한 뒤, 두 수의 곱을 최대공약수로 나누어 최소공배수를 구합니다.

## 코드
```kotlin
fun gcd(a: Int, b: Int): Int {
    return if (b == 0) a
    else gcd(b, a % b)
}

fun main() {
    val (a, b) = readln().split(" ").map { it.toInt() }

    println(gcd(a, b))
    println(a * b / gcd(a, b))
}

```
