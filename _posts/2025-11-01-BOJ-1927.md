## 링크
[1927번: 최소 힙](https://www.acmicpc.net/problem/1927)

## 풀이
`PriorityQueue`로 최소 힙을 구현합니다.

## 코드
```kotlin
import java.util.*

fun main() {
    val pq = PriorityQueue<Int>()

    repeat(readln().toInt()) {
        val x = readln().toInt()

        if (x > 0) pq.add(x)
        else println(pq.poll() ?: 0)
    }
}

```
