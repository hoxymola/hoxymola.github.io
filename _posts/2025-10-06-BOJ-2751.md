## 링크
[2751번: 수 정렬하기 2](https://www.acmicpc.net/problem/2751)

## 풀이
`StringBuilder`를 사용하여 출력 시간을 단축합니다.

## 코드
```kotlin
fun main() = with(StringBuilder()) {
    List(readln().toInt()) { readln().toInt() }
        .sorted().forEach { appendLine(it) }

    println(toString())
}

```
