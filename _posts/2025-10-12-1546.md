### 링크
[1546번: 평균](https://www.acmicpc.net/problem/1546)

### 풀이
생략

### 코드
```kotlin
fun main() {
    val n = readln().toInt()
    val scores = readln().split(" ").map { it.toDouble() }
    val m = scores.maxOrNull() ?: 0.0
    val newScores = scores.map { it / m * 100 }

    println(newScores.sum() / n)
}

```
