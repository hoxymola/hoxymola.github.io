### 링크
[30802번: 웰컴 키트](https://www.acmicpc.net/problem/30802)

### 풀이
생략

### 코드
```kotlin
fun main() {
    val n = readln().toInt()
    val sizes = readln().split(" ").map { it.toInt() }
    val (t, p) = readln().split(" ").map { it.toInt() }
    var answer = 0

    repeat(6) {
        answer += (sizes[it] - 1 + t) / t
    }

    println(answer)
    println("${n / p} ${n % p}")
}

```
