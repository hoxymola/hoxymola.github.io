## 링크
[2798번: 블랙잭](https://www.acmicpc.net/problem/2798)

## 풀이
생략

## 코드
```kotlin
fun main() {
    val (n, m) = readln().split(" ").map { it.toInt() }
    val cards = readln().split(" ").map { it.toInt() }
    var answer = 0

    for (i in 0..n - 3) {
        for (j in i + 1..n - 2) {
            for (k in j + 1..n - 1) {
                val sum = cards[i] + cards[j] + cards[k]

                if (sum <= m) {
                    answer = maxOf(answer, sum)
                }
            }
        }
    }

    println(answer)
}

```
