### 링크
[2164번: 카드2](https://www.acmicpc.net/problem/2164)

### 풀이
`ArrayDeque`을 이용하여 큐를 구현합니다.

### 코드
```kotlin
fun main() {
    val n = readln().toInt()
    val deque = ArrayDeque((1..n).toList())

    while (deque.size > 1) {
        deque.removeFirst()
        deque.addLast(deque.removeFirst())
    }

    println(deque.first())
}

```
