## 링크
[1181번: 단어 정렬](https://www.acmicpc.net/problem/1181)

## 풀이
`sortedWith`를 사용하여 정렬 기준을 직접 지정합니다.\
`a - b` -> 오름차순 정렬\
`b - a` -> 내림차순 정렬

## 코드
```kotlin
fun main() {
    val n = readln().toInt()
    val words = List(n) { readln() }

    words.distinct().sortedWith { a, b ->
        if (a.length != b.length) {
            a.length - b.length
        } else {
            a.compareTo(b)
        }
    }.forEach { println(it) }
}

```
