## 링크
[11286번: 절댓값 힙](https://www.acmicpc.net/problem/11286)

## 풀이
`PriorityQueue`로 절댓값 힙을 구현합니다.

## 코드
```kotlin
import java.util.*
import kotlin.math.abs

fun main() {
    val pq = PriorityQueue<Int>(compareBy({ abs(it) }, { it }))

    repeat(readln().toInt()) {
        val x = readln().toInt()

        if (x != 0) pq.add(x)
        else println(pq.poll() ?: 0)
    }
}

```
