## 링크
[10951번: A+B - 4](https://www.acmicpc.net/problem/10951)

## 풀이
`readLine()`이 `null`을 반환할 때까지 반복하여 입력을 처리합니다.

## 코드
```kotlin
fun main() {
    while (true) {
        val (a, b) = readLine()?.split(" ")?.map { it.toInt() } ?: return

        println(a + b)
    }
}

```
