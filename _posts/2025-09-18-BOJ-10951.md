## 링크
[10951번: A+B - 4](https://www.acmicpc.net/problem/10951)

## 풀이
`EOF` 예외가 발생할 때까지 반복하여 입력을 처리합니다.

## 코드
```kotlin
fun main() {
    while (true) {
        try {
            val (a, b) = readln().split(" ").map { it.toInt() }

            println(a + b)
        } catch (_: Exception) {
            return
        }
    }
}

```
