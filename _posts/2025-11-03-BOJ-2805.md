## 링크
[2805번: 나무 자르기](https://www.acmicpc.net/problem/2805)

## 풀이
<span class="txt_bg">매개변수 탐색(`Parametric Search`)</span>으로 해결할 수 있습니다.\
절단기의 높이를 매개변수로 두고, 조건을 만족하는 최대 높이 `m`을 찾습니다.

> [이분 탐색(Binary Search) 헷갈리지 않게 구현하기](https://www.acmicpc.net/blog/view/109)
{: .prompt-tip }

## 코드
```kotlin
lateinit var trees: List<Int>

fun check(mid: Long, k: Int): Boolean {
    return k <= trees.sumOf { maxOf(it - mid, 0) }
}

fun main() {
    val (n, m) = readln().split(" ").map { it.toInt() }
    trees = readln().split(" ").map { it.toInt() }

    var low = 0L
    var high = Int.MAX_VALUE + 1L
    while (low + 1 < high) {
        val mid = (low + high) / 2

        if (check(mid, m)) low = mid
        else high = mid
    }

    println(low)
}

```
