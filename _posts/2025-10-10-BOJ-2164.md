## 링크
[2164번: 카드2](https://www.acmicpc.net/problem/2164)

## 풀이
`Queue`를 사용하여 카드 더미를 구현한다.

## 코드
```kotlin
fun main() {
    val n = readln().toInt()
    val deque = ArrayDeque((1..n).toList())

    while (deque.size > 1) {
        deque.removeFirst()
        deque.addLast(deque.removeFirst())
    }

    println(deque.first())
}

```
