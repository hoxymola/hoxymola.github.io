## 링크
[11279번: 최대 힙](https://www.acmicpc.net/problem/11279)

## 풀이
`PriorityQueue`로 최대 힙을 구현합니다.
- `naturalOrder()`: 오름차순 정렬(기본값)
- `reverseOrder()`: 내림차순 정렬

## 코드
```kotlin
import java.util.*

fun main() {
  val pq = PriorityQueue<Int>(reverseOrder())

  repeat(readln().toInt()) {
    val x = readln().toInt()

    if (x > 0) pq.add(x)
    else println(pq.poll() ?: 0)
  }
}

```
