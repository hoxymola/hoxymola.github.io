## 링크
[15652번: N과 M (4)](https://www.acmicpc.net/problem/15652)

## 풀이
`1`부터 `N`까지의 자연수 중에서 `M`개를 선택해 <span class="txt_bg">비내림차순 수열</span>을 만드는 조합 문제입니다.

백트래킹으로 길이가 `M`인 수열을 채워 나가며,\
이전에 선택한 숫자보다 같거나 큰 수만 이어서 선택하도록 시작값을 넘겨줍니다.
- `d`: 재귀의 깊이(현재까지 채운 수열의 길이)
- `s`: 시작값(이번 단계에서 선택할 수 있는 최소 숫자)

## 코드
```kotlin
lateinit var sequence: IntArray

fun backTrack(n: Int, m: Int, k: Int, l: Int) {
    if (m == k) return println(sequence.joinToString(" "))

    (l..n).forEach {
        sequence[k] = it
        backTrack(n, m, k + 1, it)
    }
}

fun main() {
    val (n, m) = readln().split(" ").map { it.toInt() }
    sequence = IntArray(m)

    backTrack(n, m, 0, 1)
}

```
