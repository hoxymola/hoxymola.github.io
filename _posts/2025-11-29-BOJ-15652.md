## 링크
[15652번: N과 M (4)](https://www.acmicpc.net/problem/15652)

## 풀이
`N` 이하의 자연수 중에서 `M`개를 선택해 <span class="txt_bg">중복 조합</span>을 만드는 백트래킹 문제입니다.

백트래킹으로 길이 `M`인 수열을 채워 나가며,\
이전에 선택한 숫자보다 <span class="txt_bg">같거나 큰 값</span>만 선택하도록 시작 인덱스를 함께 넘겨줍니다.
- `d`: 재귀의 깊이(현재까지 채운 수열의 길이)
- `s`: 시작 인덱스(이번 단계에서 선택할 수 있는 최소 위치)

## 코드
```kotlin
lateinit var sequence: IntArray

fun backtrack(n: Int, m: Int, d: Int, s: Int) {
    if (m == d) return println(sequence.joinToString(" "))

    (s..n).forEach {
        sequence[d] = it
        backtrack(n, m, d + 1, it)
    }
}

fun main() {
    val (n, m) = readln().split(" ").map { it.toInt() }
    sequence = IntArray(m)

    backtrack(n, m, 0, 1)
}

```
